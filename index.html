what i've created...

University Database Project

For my CPSC 332 - File Strucutres & Databases course at CSUF, I led a team of three in building a mock university database. We designed the schema, creating an ER diagram as well as a relational model. The database was implemented on school servers using mySQL and MariaDB. I designed the user interface using HTML, CSS, and PHP, allowing users to access data containing information about current courses, past courses, and grade history.
